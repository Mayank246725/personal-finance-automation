# Personal Finance Automation & Visualizer

A Python-based data analytics tool designed to automate the processing of personal transaction logs, reducing manual entry errors and providing clear financial insights.

## Features
* **Automated Data Cleaning:** Cleans column headers by removing trailing spaces and formats raw transaction data.
* **Expense Categorization:** Groups transactions by category (e.g., Rent, Food, Groceries, Entertainment) using **Pandas**.
* **Data Visualization:** Generates a clean, customized bar chart of monthly expenses using **Matplotlib** and **Seaborn**.
* **Auto-Save:** Saves the visual breakdown automatically as an image (`expense_chart.png`).

## Technologies Used
* **Python 3.13**
* **Pandas** (Data Manipulation)
* **Matplotlib & Seaborn** (Data Visualization)
